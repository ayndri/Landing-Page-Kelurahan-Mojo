01

Latar Belakang

Kondisi Sebelumnya

✗ Profil RW (ketua, visi-misi, data KK) hanya di papan fisik
✗ Potensi tanaman & TOGA warga tidak terdokumentasi
✗ UMKM belum terpromosikan secara digital
✗ Pengumuman hanya via surat & grup WA, sering terlewat
✗ Layanan administrasi tidak ada panduan online

Solusi: Website Profil Digital

✓ Profil digital per-RW lengkap dengan pengurus
✓ Direktori tanaman + foto + peta + QR Code
✓ Direktori UMKM + produk + order via WhatsApp
✓ Pengumuman & agenda digital terkelola resmi
✓ Info kontak & layanan administrasi kelurahan

Cara Menggunakan Website

Untuk Warga
1. Buka kelurahan-mojo.com di HP / laptop
2. Pilih RW Anda → lihat profil & potensi
3. Jelajahi Tanaman, UMKM, atau Peta
4. Baca Pengumuman & cek Agenda Kegiatan
5. Pesan produk UMKM via tombol WhatsApp
6. Scan QR Code di taman / kemasan produk
7. Cek syarat layanan surat di halaman Kontak
Untuk Admin RW
1. Buka kelurahan-mojo.com/admin/login
2. Masukkan email & password akun RW
3. Dashboard: lihat statistik data RW
4. Tambah / edit Tanaman, UMKM, Galeri
5. Buat Pengumuman & Agenda baru
6. Edit Profil RW, Struktur Pengurus
7. Pilih lokasi tanaman/UMKM di peta klik
